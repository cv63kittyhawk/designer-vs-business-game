# Designer vs Business

単一HTMLで作成したゲームプロトタイプです。

## 概要

「UX（ユーザーゴール）」と「売上（ビジネスゴール）」のトレードオフをテーマにした意思決定シミュレーションゲームです。
プレイヤーはプロダクト開発の現場に立ち、毎ターン提示される課題に対して2択の判断を行います。

選択によって以下の3つの指標が変動します：

* ユーザー信頼（UX）
* 売上（ビジネス）
* チーム健全性

最終的な数値バランスによってエンディングが分岐し、プレイヤーの意思決定傾向が可視化されます。

## 遊び方

1. ページを開く
2. 各ターンで提示される選択肢を選ぶ
3. 数値の変化を見ながら進める
4. 最終ターンまで進むと結果が表示される

## 目的

このプロトタイプでは以下を検証することを目的としています：

* UXと売上の対立構造をゲームとして体験可能にできるか
* 意思決定の積み重ねによってプレイヤーの思考傾向を可視化できるか
* テキストベースでも「ゲームとして成立する体験」を作れるか
* 単一HTMLでどこまで構造的なシミュレーションを実装できるか

It’s not about right or wrong, it’s about trade-offs.（正しさではなく、トレードオフの設計がテーマ）
